    TiledMap¶

    tiledpy.TiledMap is the main entry point. It parses a .tmx file and exposes layers, tilesets, and rendering methods.

    from tiledpy import TiledMap

tmap = TiledMap("map.tmx")

    tiledpy.loader.TiledMap ¶

    A Tiled map loaded from a .tmx file.

    Parses the TMX XML, builds :class:~tiledpy.tileset.Tileset, :class:~tiledpy.layer.TileLayer, and :class:~tiledpy.layer.ObjectLayer objects, and exposes rendering helpers for Pygame.

    Parameters:

    Name Type Description Default
    path str

    Absolute or relative path to the .tmx file.

    		 required

    Attributes:

    Name Type Description
    path str

    Absolute path to the .tmx file.
    base_dir str

    Directory containing the .tmx (used to resolve relative paths to images and external tilesets).
    orientation str

    Map orientation: "orthogonal", "isometric", etc.
    render_order str

    Tile render order, e.g. "right-down" (default).
    width int

    Map width in tiles.
    height int

    Map height in tiles.
    tile_width int

    Tile width in pixels.
    tile_height int

    Tile height in pixels.
    infinite bool

    True if the map uses chunk-based infinite scrolling.
    background_color str or None

    Hex background color string "#rrggbb", or None.
    tilesets list[Tileset]

    All tilesets sorted by firstgid ascending.
    layers list[TileLayer or ObjectLayer]

    All layers in draw order.
    properties dict

    Custom map-level properties.

    Raises:

    Type Description
    FileNotFoundError

    If any tileset image referenced by the TMX/TSX is missing.

    Examples:

    >>> tmap = TiledMap("map.tmx")
>>> tmap.draw_layer(screen, "ground", offset=(cam_x, cam_y))

    get_tile_gid(tx, ty, layer_name=None) ¶

    Return the raw GID at tile coordinates (tx, ty).

    Parameters:

    Name Type Description Default
    tx int

    Tile X coordinate.

    		 required
    ty int

    Tile Y coordinate.

    		 required
    layer_name str or None

    Name of the layer to query. If None, all :class:~tiledpy.layer.TileLayer instances are searched in draw order and the first non-zero GID is returned.

    		 None

    Returns:

    Type Description
    int

    Raw GID (may include flip/rotate flags). Returns 0 if the tile is empty or the layer does not exist.

    Examples:

    >>> gid = tmap.get_tile_gid(3, 5, "ground")
>>> gid = tmap.get_tile_gid(3, 5)          # first non-empty layer

    get_tileset_for_gid(gid) ¶

    Return the Tileset that owns the given global tile ID.

    Uses a linear scan (tilesets are sorted by firstgid), returning the last tileset whose firstgid is less than or equal to gid.

    Parameters:

    Name Type Description Default
    gid int

    Global tile ID (without flip flags).

    		 required

    Returns:

    Type Description
    Tileset or None

    The owning tileset, or None if no tileset covers this GID.

    Examples:

    >>> ts = tmap.get_tileset_for_gid(42)
>>> local = ts.global_to_local(42)

    draw_layer(surface, layer_name, offset=(0, 0), scale=1) ¶

    Render one named TileLayer onto a pygame.Surface.

    Delegates to :func:tiledpy.renderer.draw_layer. Tiles outside the surface bounds are skipped (viewport culling). Does nothing if the layer does not exist or is not a :class:TileLayer.

    Parameters:

    Name Type Description Default
    surface Surface

    Render target.

    		 required
    layer_name str

    Name of the layer to draw.

    		 required
    offset tuple[int, int]

    Camera offset (ox, oy) in pixels, by default (0, 0).

    		 (0, 0)
    scale int

    Integer scale factor for pixel-art rendering, by default 1.

    		 1

    Examples:

    >>> tmap.draw_layer(screen, "water", offset=(cam_x, cam_y), scale=2)

    Supported TMX features¶

    Feature Supported
    Finite maps Yes
    Infinite maps (chunks) Yes
    CSV encoding Yes
    Base64 + zlib Yes
    Base64 + gzip Yes
    Base64 + zstd Yes (pip install zstandard)
    External .tsx tilesets Yes
    Inline tilesets Yes
    Tile properties Yes
    Tile collision objects Yes
    Tile animations Yes (data stored, no auto-play)
    Object layers Yes
    Flip / rotate flags Yes
    Group layers No
    Image layers No
